
# WorkWave – Employee Management System

## Overview

WorkWave is a web-based Employee Management System developed using PHP. It is designed to streamline employee-related operations such as task assignment, attendance tracking, salary management, and role-based access.

The system supports multiple user roles including Admin, HR, and Employee, ensuring efficient workflow management and secure access control.

---

## Project Objective

The objective of this project is to:

- Manage employee records and activities  
- Assign and track tasks efficiently  
- Monitor attendance and work status  
- Handle salary confirmation and payslips  
- Implement role-based access control  

---

## Features

### Role-Based Access

- **Admin**
  - Manage employees and system operations  
  - Assign tasks to employees  
  - Monitor overall performance  

- **HR**
  - Manage employee attendance  
  - Confirm and manage salaries  
  - Handle employee records  

- **Employee**
  - View assigned tasks  
  - Update task status  
  - Check attendance  
  - View salary slips  

---

### Task Management

- Assign tasks to employees  
- Track task progress and status updates  
- Monitor completion of tasks  

---

### Attendance Management

- Mark and view attendance  
- Maintain employee attendance records  

---

### Salary Management

- HR can confirm employee salary  
- Employees can view their salary slips

## How to Run

1. Install XAMPP / WAMP / LAMP  
2. Place the project folder inside `htdocs`  

3. Start Apache and MySQL  

4. Import the database in phpMyAdmin  

5. Open browser and run:

```

[http://localhost/WorkWave](http://localhost/WorkWave)

```
